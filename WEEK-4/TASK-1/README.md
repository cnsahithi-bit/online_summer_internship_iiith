# TASK-1

In this task, we created a Python virtual environment using venv and installed the Ultralytics package to work with YOLOv8 models. We trained an object detection model using the command `yolo detect train model=yolov8n.pt data=coco128.yaml epochs=5 imgsz=320 batch=8`, which automatically downloaded the dataset and generated performance metrics like precision, recall, and mAP. We explored the YOLO dataset structure, which includes images/ and labels/ folders divided into train/ and val/, where each image has a corresponding .txt label file with the same name. These labels files store object annotations such as class IDs and coordinates—bounding boxes for detection and polygon points for segmentation datasets like coco128-seg. We also understood the role of the data.yaml file, which defines dataset paths, number of classes, and class names.
